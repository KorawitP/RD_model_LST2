# 3.4 สถาปัตยกรรมแบบจำลอง (Model Architecture)

การวิจัยนี้เลือกใช้ **Random Forest Regressor** เป็นอัลกอริทึมหลักในการพัฒนาแบบจำลอง เนื่องจากเป็นอัลกอริทึมประเภท Ensemble Learning ที่มีความสามารถสูงในการจัดการกับข้อมูลที่มีความซับซ้อนและความสัมพันธ์แบบไม่เป็นเชิงเส้น (Non-linear relationships) ซึ่งเป็นลักษณะทั่วไปของข้อมูลทางอุตุนิยมวิทยาและสิ่งแวดล้อม อีกทั้งยังมีความทนทานต่ออคติ (Bias) และลดความเสี่ยงของการเกิดภาวะการเรียนรู้เกิน (Overfitting) ได้ดีกว่า Decision Tree เดี่ยว

ในการพัฒนาแบบจำลอง ได้มีการกำหนดค่าไฮเปอร์พารามิเตอร์ (Hyperparameters) ที่เหมาะสมที่สุดผ่านการทดสอบ เพื่อให้ได้ประสิทธิภาพสูงสุด ดังนี้:

| พารามิเตอร์ (Parameter) | ค่าที่กำหนด (Value) | คำอธิบาย (Description)                                                             |
| :---------------------- | :------------------ | :--------------------------------------------------------------------------------- |
| **n_estimators**        | 100                 | จำนวนต้นไม้การตัดสินใจ (Decision Trees) ในป่าแบบสุ่ม                               |
| **max_depth**           | 25                  | ความลึกสูงสุดของต้นไม้แต่ละต้น ถูกจำกัดไว้เพื่อป้องกัน Overfitting                 |
| **min_samples_split**   | 20                  | จำนวนตัวอย่างขั้นต่ำที่ต้องมีเพื่อทำการแตกกิ่ง (Split a node)                      |
| **min_samples_leaf**    | 10                  | จำนวนตัวอย่างขั้นต่ำที่ต้องมีในใบ (Leaf node) สุดท้าย                              |
| **max_features**        | 'sqrt'              | จำนวนคุณลักษณะที่พิจารณาในการแตกกิ่งแต่ละครั้ง (รากที่สองของจำนวนคุณลักษณะทั้งหมด) |
| **random_state**        | 42                  | ค่าตั้งต้นสำหรับการสุ่ม เพื่อให้ผลการทดลองมีความคงที่                              |

แบบจำลองถูกพัฒนาโดยใช้ภาษา Python ผ่านไลบรารี **Scikit-learn** ซึ่งเป็นเครื่องมือมาตรฐานสำหรับการเรียนรู้ของเครื่อง
