# 3.1 ภาพรวมของวิธีการดำเนินงานวิจัย

ในการศึกษานี้ ผู้วิจัยได้กำหนดขั้นตอนการดำเนินงานเพื่อพัฒนาแบบจำลองการเรียนรู้ของเครื่อง (Machine Learning Model) สำหรับทำนายอุณหภูมิพื้นผิวดิน (Land Surface Temperature: LST) ของประเทศไทย โดยมีขั้นตอนหลักทั้งหมด 4 ขั้นตอน ดังแสดงในภาพรวมกระบวนการ (Workflow) ต่อไปนี้

1.  **การรวบรวมข้อมูล (Data Acquisition):** การดาวน์โหลดและรวบรวมข้อมูลเชิงพื้นที่และอนุกรมเวลา (Spatial and Temporal Data) จากแพลตฟอร์ม Google Earth Engine (GEE) ซึ่งประกอบด้วยข้อมูลอุณหภูมิพื้นผิว, ก๊าซเรือนกระจก, ดัชนีพืชพรรณ และข้อมูลสภาพประกอบอื่นๆ
2.  **การเตรียมและประมวลผลข้อมูล (Data Preprocessing):** ขั้นตอนการจัดการคุณภาพข้อมูล ได้แก่ การจัดการค่าที่หายไป (Missing Data Imputation), การสุ่มตัวอย่างข้อมูล (Data Sampling), และการคัดเลือกคุณลักษณะ (Feature Selection) เพื่อให้ได้ชุดข้อมูลที่เหมาะสมที่สุดสำหรับการฝึกสอนแบบจำลอง
3.  **การพัฒนาและฝึกสอนแบบจำลอง (Model Development & Training):** การสร้างและตั้งค่าพารามิเตอร์สำหรับแบบจำลอง Random Forest Regressor และดำเนินการฝึกสอน (Training) ด้วยชุดข้อมูลที่เตรียมไว้
4.  **การประเมินประสิทธิภาพ (Model Evaluation):** การวัดผลการทำงานของแบบจำลองโดยใช้ตัวชี้วัดทางสถิติ ได้แก่ ค่าสัมประสิทธิ์การตัดสินใจ ($R^2$) และรากที่สองของค่าเฉลี่ยความคลาดเคลื่อนกำลังสอง (RMSE) รวมถึงการตรวจสอบความเสถียรของแบบจำลองด้วยวิธี Cross-Validation

กระบวนการทั้งหมดนี้ถูกออกแบบมาเพื่อให้มั่นใจว่าผลลัพธ์ที่ได้มีความถูกต้อง น่าเชื่อถือ และสามารถนำไปประยุกต์ใช้ในการวิเคราะห์สภาพภูมิอากาศของประเทศไทยได้อย่างมีประสิทธิภาพ
